# Bedienungsanleitung für das Hansa Homerenovierung CMS

Willkommen zu Ihrem neuen Content Management System (CMS). Mit diesem Tool können Sie die Inhalte Ihrer Website einfach und ohne technische Vorkenntnisse selbst verwalten.

---

## 1. Anmeldung am System

Um Änderungen vorzunehmen, müssen Sie sich zuerst anmelden:

1.  Rufen Sie die Adresse **[https://hansa-homerenovierung.com/login](https://hansa-homerenovierung.com/login)** auf.
2.  Geben Sie Ihre Zugangsdaten ein:
    -   **Benutzer:** `admin`
    -   **Passwort:** `password123`
3.  Klicken Sie auf **"Anmelden"**.

---

## 2. Verwaltung der Leistungen

Hier können Sie die Dienstleistungen bearbeiten, die auf Ihrer Website unter dem Punkt "Leistungen" erscheinen.

1.  Klicken Sie im Dashboard auf **"Leistungen"**.
2.  **Neue Leistung erstellen:** Klicken Sie oben auf "Leistung hinzufügen".
3.  **Bestehende Leistung bearbeiten:** Klicken Sie in der Liste auf die entsprechende Leistung.
4.  **Inhalte anpassen:**
    -   **Titel:** Der Name der Dienstleistung (z.B. "Bad-Sanierung").
    -   **Beschreibung:** Ein ausführlicher Text über Ihre Arbeit.
    -   **Bild:** Wählen Sie ein Foto von Ihrem Computer aus. Das System optimiert das Bild automatisch für schnelle Ladezeiten.
5.  Klicken Sie auf **"Speichern"**.

---

## 3. Verwaltung der Referenzen (Projekte)

Hier zeigen Sie Ihren Kunden, welche Arbeiten Sie bereits erfolgreich abgeschlossen haben.

1.  Klicken Sie im Dashboard auf **"Referenzen"**.
2.  Gehen Sie genauso vor wie bei den Leistungen:
    -   Geben Sie den **Kunden** oder den **Ort** an.
    -   Fügen Sie ein **Vorschaubild** hinzu.
    -   Beschreiben Sie kurz das Projekt (z.B. "Komplettrenovierung einer Altbauwohnung").

---

## 4. Tipps für Bilder

Damit die Website schnell bleibt und professionell aussieht, beachten Sie bitte:

-   **Formate:** Sie können `.jpg`, `.png` oder `.webp` Dateien hochladen.
-   **Größe:** Ein einzelnes Bild sollte nicht größer als **5 MB** sein.
-   **Optimierung:** Sie müssen sich nicht um das Verkleinern von Bildern kümmern – das CMS übernimmt das automatisch für Sie.

---

## 5. Kontaktanfragen

Wenn ein Kunde das Kontaktformular auf Ihrer Website ausfüllt:

1.  Erhalten Sie automatisch eine E-Mail an **mohsenturkey12@gmail.com**.
2.  In dieser E-Mail finden Sie den Namen, die E-Mail-Adresse und die Nachricht des Kunden.
3.  Sie können direkt auf diese E-Mail antworten, um mit dem Kunden Kontakt aufzunehmen.

---

## 6. Wichtige Hinweise

-   **Passwort-Sicherheit:** Geben Sie Ihre Login-Daten nicht an Unbefugte weiter.
-   **Vorschau:** Nach dem Speichern im CMS sind die Änderungen sofort live für alle Besucher der Website sichtbar.

Sollten Sie Fragen haben oder Unterstützung benötigen, schauen Sie in die technischen Unterlagen (`DEPLOYMENT.md`) oder kontaktieren Sie Ihren Administrator.
